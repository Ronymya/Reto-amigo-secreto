Descripción

Este proyecto es una aplicación web sencilla que permite a los usuarios:

Agregar nombres de amigos a una lista.

Visualizar la lista actualizada en la interfaz.

Realizar un sorteo aleatorio para elegir un "amigo secreto" de entre los nombres ingresados.

Es ideal para reuniones, juegos y eventos sociales donde se necesite un sistema para sortear participantes.

Características

Interfaz de Usuario:

Campo de texto para ingresar los nombres de los amigos.

Botón para agregar nombres a la lista.

Visualización dinámica de los nombres en una lista interactiva.

Botón para realizar el sorteo del amigo secreto.

Validaciones:

Evita agregar nombres vacíos.

Verifica que haya al menos dos nombres antes de realizar el sorteo.
